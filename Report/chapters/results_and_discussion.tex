\chapter{Results and Discussion}

This chapter presents the experimental results and analysis of the proposed Quantum Retrieval-Augmented Generation (QRAG) framework. The objective is to evaluate the effectiveness of the hybrid quantum-classical approach in resolving syntactic ambiguity and improving system performance. :contentReference[oaicite:0]{index=0}

\section{Experimental Setup}

The evaluation was conducted using a curated Ambiguity Challenge Dataset consisting of 15 syntactically complex queries, including garden path sentences, prepositional phrase attachment, reduced relative clauses, and gerund ambiguity cases.

The user interface used for executing queries and visualizing results is shown in Figure~\ref{fig:website_homepage}.

\section{Evaluation Metrics}

The performance was evaluated using the following metrics:
\begin{itemize}
    \item Context Relevance (CR)
    \item Answer Faithfulness (AF)
    \item Answer Relevance (AR)
\end{itemize}

\section{Aggregate Performance Analysis}

Out of the 15 test queries:
\begin{itemize}
    \item 10 queries showed improvement using the quantum pipeline
    \item 3 queries showed similar performance
    \item 2 queries showed reduced performance due to quantum noise
\end{itemize}

These results confirm that quantum advantage is dependent on query complexity.

\section{Discussion}

The results demonstrate that:
\begin{itemize}
    \item Classical systems are efficient but limited in handling ambiguity
    \item Quantum systems provide improved semantic understanding for complex queries
    \item Hybrid architecture provides an optimal balance between performance and efficiency
\end{itemize}

\bigskip\noindent
Overall, the QRAG framework successfully enhances semantic interpretation in ambiguous queries while maintaining efficiency for simpler cases. The results validate the effectiveness of integrating quantum processing into retrieval-augmented generation systems.
